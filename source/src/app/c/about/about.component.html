Warum nicht die App?
  • Keine Sicherheitsprobleme (wie hart gecodete Passwörter)
  • Bessere Visuals
  • Progressive Web App
    • Multiplatform Support (Apples iOS, Android, Windows[Phones], OsX, Linux, sogar dein f*cking Fernseher oder Auto)
    • Support für alte Geräte (solange der Browser nicht zu alt ist)
  • Kurs Cloud
  • Easy2Use
Sind meine Daten sicher?
format_quote

Ja. Um CORS Verbote zu umgehen verwende ich ein Proxy Script, welches die angefragten Daten kurz zwischengespeichert. Der Code dafür ist ebenfalls im Github Respo. Deine in der Kurscloud angegebenen Daten sind es nicht! Das System ist autorisierungsfrei aufgebaut, keine Nutzer, keine Passwörter. Das löschen dieser ist nicht mehr möglich, da es keine Möglichkeit gibt zu Beweisen, dass du die ID registriert hast. (Das überschreiben aber auch nicht).

Wo werden meine Daten gespeichert?
format_quote

Ich verwende Firebase Realtime DB um die Kursdaten zu speichern. Ist das nicht unsicher? Nein.
