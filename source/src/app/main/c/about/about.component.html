FAQ

Wer hat das gemacht?

Ich.

Gibt es irgendetwas was ich wissen muss?

Die Kurs Cloud wir zu jedem Halbjahr zurückgesetzt.

Wie viel mobile Daten werden verbraucht?

Beim ersten laden (inklusive Setup von Kursen etc.) werden in etwa 1.6 MB verbraucht. Danach sollten nur etwa 40 KB anfallen!

Brauch ich ständig internet?

Nein. Für eine Verbindung zu den Vertretungsdaten natürlich schon.

Warum nicht die App?

  • Keine Sicherheitsprobleme (wie hart gecodete Passwörter)
  • Bessere Visuals
  • Multiplatform Support (iOS, Android, Windows[Phones], OsX, Linux, sogar dein f*cking Fernseher)
  • Kurs Cloud

Sind meine Daten sicher?

Ja. Um CORS Verbote zu umgehen verwende ich ein Proxy Script, welches die angefragten Daten kurz zwischengespeichert. Der Code dafür ist ebenfalls im Github Repo. Deine in der Kurscloud angegebenen Daten sind es nicht! Das System ist autorisierungsfrei aufgebaut, keine Nutzer, keine Passwörter. Das löschen dieser ist nicht mehr möglich, da es keine Möglichkeit gibt zu Beweisen, dass du die ID registriert hast. (Das überschreiben aber auch nicht).

Wo werden meine Daten gespeichert?

Ich verwende Firebase Realtime DB um die Kursdaten zu speichern. Ist das nicht unsicher? Nein.
