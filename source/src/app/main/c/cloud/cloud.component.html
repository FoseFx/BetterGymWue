Gar nicht. Das System ist so aufgebaut, dass jede ID genau einmal gesetzt werden kann und dann nicht mehr bearbeitet werden kann. Dies hat den Vorteil, dass man keine Verifizierung (User/Psw) benötigt. Um dir eine neue ID zu generieren, kannst du deine Kurse löschen und neu eingeben.

Kurs Cloud

So wie es aussieht, verwendest du die Kurs-Cloud nicht. Sollte das falsch sein, lösche deine Kurse und gibt deine Kurs Cloud ID im Konfigurationsprozess ein. Wenn du die Kurs Cloud mit deiner aktuellen Konfiguration aktivieren möchtest, kannst du das mit nur einem Klick. Du bekommst dann einen Code, welchen du überall (iOs, Android, Windows, sprich X-Platform) verwenden kannst, um deine Kurse nicht immer neu eingeben zu müssen. Die Kurs-Cloud wird jedes Halbjahr zurückgesetzt.
