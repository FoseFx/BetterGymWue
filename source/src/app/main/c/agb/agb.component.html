Welche Informationen werden gesammelt? Und warum?

Jede Anfrage, die auf einen Server von mir getätigt wird schickt dein Browser deine angefragte URL (bspw. "https://proxy.fosefx.com/v2") gesetzte Cookies (es werden keine Cookies gesetzt) den "User-Agent" und andere Http-Header mit. Letzterer ist ein Kurzer Text, der deinen Browser identifiziert (bspw. "Mozilla/5.0 (Windows NT 6.1; rv:60.0) Gecko/20100101 Firefox/60.0") ("Logdaten"). Bevor die Anfrage an meinem Server ankommt durchläuft sie Cloudflare (Cloudflare, Inc., 101 Townsend St, San Francisco, CA 94107, USA) Server. All diese Daten werden von mir und Cloudflare analysiert um Risiken für den Server zu erkennen und zu blockieren. Für mehr Informationen zu Cloudflare: https://www.cloudflare.com/privacypolicy/. Für dich relevant sind die "End User" betreffenden Punkte.
Darüber hinaus werden, wenn du zustimmst (Normalfall), Informationen über deine Kurse auf den Servern von Google (Google LLC, 1600 Amphitheatre Parkway, Mountain View, CA 94043, USA ) für ein Halbjahr gespeichert um die Kurs Cloud zu realisieren.
Diese Seite verwendet Google Fonts, ein Service von Google für Schriftarten. Dafür fragt der Browser bei Servern von Google an und schickt Logdaten mit.
Solltest du das Hilfe Formular verwenden, werden auch diese Daten auf Google Servern gepeichert um sie später dort aubrufen und verarbeiten zu können.
Nebenbei: Bei Kontakt direkt an eine fosefx.com Adresse ("max@fosefx.com", "info@fosefx.com") wird deine Email verschlüsselt auf den Servern der Proton Technologies AG (Chemin du Pré-Fleuri, 3 CH-1228 Plan-les-Ouates, Genv, Schweiz) gespeichert. Vermutlich auch unverschlüsselt auf Servern deines Email Anbieters.

Die oben genannten Anbieter erhalten die Informationen auf sicherem Weg (https/TLS verschlüsselte Verbindung). Dabei werden die Daten nicht verkauft, sondern dienen nur den oben beschriebenen Gründen.

Welche Kontrolle hat ein Nutzer über seine Daten?

Wenig. Da es keine Mölichkeit gibt zu beweisen, dass jemand zu seinen Daten gehört ist es nicht möglich Daten auf Anfrage zu löschen.

Wie sicher sind die Daten?

Alle bei Google gespeicherten Daten liegen auf Servern, die ISO 27001, SOC 1, SOC 2, und SOC 3 beständigt sind. Daten stehen im Falle von Nutzung US-amerikanischer Servern unter dem Schutz des Privacyshield Frameworks. Dies gilt sowohl für Cloudflare als auch Google. Mein Server gehört zu einem Rechenzentrum in Limburg, Deutschland.
